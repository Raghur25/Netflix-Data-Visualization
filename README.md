Netflix Data Visualization

This project analyzes and visualizes the Netflix Movies and TV Shows dataset using Python, Pandas, and Matplotlib.
All work was done in VS Code, and the project includes multiple visual charts that reveal trends in Netflix content.

What This Project Does:

Loads and cleans the Netflix dataset
Handles missing values
Visualizes important insights using Matplotlib
Saves all charts inside the project folder

Visualizations included:
Movies vs TV Shows (Bar Chart)
Content Rating Percentages (Pie Chart)
Distribution of Movie Durations (Histogram)
Release Year vs Number of Shows (Scatter Plot)
Top 10 Countries with Most Content (Horizontal Bar Chart)
Movies vs TV Shows Over Time (Subplots)


Technologies Used

Python
Pandas
Matplotlib
VS Code


How to Run the Project (VS Code)
1. Install dependencies
pip install pandas matplotlib
2. Run the script
python netflix_analysis.py
